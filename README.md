# Windchill [![Netlify Status](https://api.netlify.com/api/v1/badges/649c58f9-6105-45ed-bcef-924441695ad3/deploy-status)](https://app.netlify.com/sites/windchill/deploys)

![Windchill Cover](https://i.ibb.co/ZSxsBws/windchillbanner.png)

Windchill is a simple wind chill calculator! 
Wind chill is what tempurature it feels like outside, accounting for wind speed as well as base tempurature.
